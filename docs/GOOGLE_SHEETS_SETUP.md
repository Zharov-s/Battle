# Настройка Google Sheets

На каждом из трех листов первая строка должна содержать точные заголовки:

`Дата оплаты | ФИО МПП | Сумма продажи | Продукт продажи`

## Рекомендованные проверки

- Дата оплаты: проверка «действительная дата», формат `ДД.ММ.ГГГГ`.
- Сумма продажи: число больше 0, формат рублей.
- На листе `Допродажи`: выпадающий список `МКБ, МОБ, РКЛ, Проверка гостей, iiko`.
- Закрепить первую строку.
- Защитить строку заголовков и ячейки миссий.

Формулы для пользовательского контроля должны оставаться короткими и понятными. Например:

- факт команды: `=СУММ(C2:C)`
- выполнение: `=ЕСЛИОШИБКА(Факт/Миссия;0)`
- остаток: `=МАКС(Миссия-Факт;0)`

Дашборд не зависит от этих контрольных формул и сам проверяет строки перед расчетом.
