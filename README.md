# МОНОПОЛИЯ — интерактивный дашборд

Дашборд для команд Шкильнюка (Гранд-Сити) и Китаевой (Ривьера-Сити). Все показатели формируются из Google Sheets и обновляются каждые 5 минут.

## Источник данных

Таблица: https://docs.google.com/spreadsheets/d/15qvaEu69wIN5Y8N6G3rBCgg39xx3enL_1lTvxZniIHk/edit

Обязательные колонки на листах `Китаева`, `Шкильнюк`, `Допродажи`:

1. `Дата оплаты`
2. `ФИО МПП`
3. `Сумма продажи`
4. `Продукт продажи`

На командных листах можно оставить миссию в отдельных ячейках. Рабочие значения миссий хранятся централизованно в `config.js` и `apps-script/Code.gs`: 2 500 000 ₽ и 2 000 000 ₽.

## Запуск без Apps Script

По умолчанию дашборд читает три листа через Google Visualization API. Для этого у таблицы должен быть доступ: **Все, у кого есть ссылка — читатель**.

## Надежный вариант через Apps Script

1. В таблице откройте `Расширения → Apps Script`.
2. Скопируйте `apps-script/Code.gs` в `Code.gs`.
3. `Deploy → New deployment → Web app`.
4. `Execute as: Me`, доступ — подходящий для вашей организации или `Anyone with the link`.
5. Скопируйте URL `/exec` и вставьте в `config.js` в `API_URL`.
6. При обновлении скрипта создавайте новую версию deployment.

## Правила расчета

- Командный факт — сумма валидных строк соответствующего командного листа.
- Лидер — команда с большим процентом выполнения собственной миссии.
- 1 отель = 100 000 ₽.
- Гранд-Сити — 20 базовых отелей, Ривьера-Сити — 25.
- Личный топ-5 считается только по листу `Допродажи` и продуктам: МКБ, МОБ, РКЛ, Проверка гостей, iiko.
- Период: 20.07.2026–31.08.2026, часовой пояс Europe/Moscow.

## Дизайн и источники компонентов

Визуальная система адаптирует паттерны карточек и индикаторов из Uiverse, паттерн activity gauge из Untitled UI и официальный Apache ECharts для диаграммы. Полные исходные ссылки перечислены в `ATTRIBUTION.md`.
